Example Requests

1. Start a Shopify to Snowflake Connection

POST /api/ai/conversation/start Content-Type: application/json { "description": "Connect Shopify to Snowflake" }

2. Continue Conversation with Store URL

POST /api/ai/conversation/continue Content-Type: application/json { "conversationHistory": [ { "role": "user", "content": "Connect Shopify to Snowflake" }, { "role": "assistant", "content": "I'll help you connect Shopify to Snowflake. What's your Shopify store URL?" } ], "answer": "My store URL is example.myshopify.com" }

4. WebSocket Conversation Continue
